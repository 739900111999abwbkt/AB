# AirChat — التطبيق الكامل

هذا المشروع يحتوي على:
- backend: كود Node.js و Express و MongoDB
- frontend: كود React أو واجهة HTML/JS جاهزة
- مهيأ بالكامل للعمل محلياً أو على سيرفر

## خطوات التشغيل:

### backend:
```bash
cd backend
npm install
npm start
```

### frontend:
```bash
cd frontend
npm install
npm start
```
